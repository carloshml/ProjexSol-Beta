O que é ProjexSol

O Projexsol é uma ferramenta facilitadora para a criação de projetos solidários que aproximem a comunidade Palmense para o meio acadêmico através da criação de projetos solidários

Sobre o Projeto

O projeto foi idealizado durante as aulas de Projeto de Sistemas do Curso de Ciência da Computação durante o semestre 2015.2 com o intuito de fornecer meios para que a UFT integre a comunidade ao meio acadêmico
